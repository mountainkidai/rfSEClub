# Git Commands

## Phase 5: Git Stash

To switch the branch without committing the current branch:

```bash
git stash
```

To pop the stash:

```bash
git stash pop
```

To copy the stash:

```bash
git stash apply
```

To list the stash list:

```bash
git stash list
```

To delete a specific stash:

```bash
git stash drop stash@{0}
```

To delete all stash:

```bash
git stash clear
```
