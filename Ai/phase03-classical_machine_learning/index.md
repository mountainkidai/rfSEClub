# Phase 3 — Classical Machine Learning

## Introduction

Traditional programming works like this:

```text
Rules
+
Data
↓
Answers
```

Machine Learning reverses the process:

```text
Data
+
Answers
↓
Rules
```

The machine discovers the rules.

This idea changed computing forever.

Machine learning is fundamentally:

```text
Experience
↓
Pattern Discovery
↓
Prediction
```

This phase teaches the fundamental algorithms that existed before deep learning.

Many modern AI systems still rely on these ideas.
