# Phase 11 — Agent Systems

## Introduction

A language model predicts tokens.

An agent achieves objectives.

This distinction is fundamental.

A model answers:

```text
What should be said next?
```

An agent answers:

```text
What should be done next?
```

Agents introduce a new loop:

```text
Observe
↓
Reason
↓
Plan
↓
Act
↓
Observe
```

This loop is the foundation of autonomous systems.

The purpose of this phase is understanding how intelligence becomes action.
