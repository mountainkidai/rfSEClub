#### 4 Simple Rules

| Your Work                       | Use This                 | Why                            |
| ------------------------------- | ------------------------ | ------------------------------ |
| Read only (reports, dashboards) | Repeatable Read          | Frozen picture all transaction |
| Read + Write (normal apps)      | Read Committed (default) | Fast + safe enough             |
| Super busy app (1000 users/sec) | Read Committed           | Maximum speed                  |
| Bank/money critical             | Serializable             | Perfect order (slow)           |
