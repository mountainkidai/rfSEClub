### Topics

APIs and Client-Server Basics – What APIs are and why client-server separation matters.

What is REST – Core principles, HTTP methods, and resource model.

Why REST Became Standard – Advantages like scalability and statelessness.

REST Limitations – Over-fetching, under-fetching, and N+1 problems.

What is GraphQL – History, single endpoint, and query language approach.

GraphQL vs REST – Key differences in data fetching and flexibility.

GraphQL Core Concepts – Operations (query/mutation), variables, field selection.

Hasura-Specific Features – Where clauses, ordering, pagination.

Implementing in Next.js – GQL helper function and TypeScript usage.

Practical Roadmap – Testing in GraphiQL and replacing REST calls.
