Every invention.

Every company.

# 0.1 What Is A Problem?

## Why This Matters

Before solving problems we must define them.

Topics:

### 0.1.1 Current State

### 0.1.2 Desired State

### 0.1.3 The Gap

### 0.1.4 Constraints

### 0.1.5 Objectives

### 0.1.6 Outcomes

### 0.1.7 Problem Definition

---

A problem exists when:

```text
Current State
≠
Desired State
```

Example:

```text
Current State:
Hungry

Desired State:
Not Hungry

Gap:
Food Needed
```

The gap is the problem.
