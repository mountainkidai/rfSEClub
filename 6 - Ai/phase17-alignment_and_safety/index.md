# Phase 17 — General Intelligence Systems

## Introduction

Most AI today is narrow.

A language model can generate text.

A vision model can analyze images.

A recommendation system can recommend content.

Humans are different.

A human can:

```text
Learn Mathematics

Write Software

Drive Vehicles

Start Companies

Conduct Research

Teach Others

Create New Fields
```

This capability is called:

```text
General Intelligence
```
