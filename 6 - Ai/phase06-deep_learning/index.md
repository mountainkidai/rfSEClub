# Phase 6 — Deep Learning

## Introduction

A single neuron is weak.

A small neural network is useful.

A deep neural network can learn abstractions.

The central discovery of Deep Learning is:

```text
Intelligence emerges from hierarchy.
```

Reality itself is hierarchical.

For example:

```text
Pixels
↓
Edges
↓
Shapes
↓
Objects
↓
Scenes
```

Language is hierarchical:

```text
Characters
↓
Words
↓
Phrases
↓
Sentences
↓
Ideas
```

Deep Learning works because it learns these hierarchies automatically.

This phase teaches how modern AI learns representations of reality.
